Ep 2. 中文编程了解一下？

2021-07-16 podcast

Astrian 一直吐槽白羊在写代码的时候使用中文变量名，因此白羊就在这一期中狠狠地向大家安利中文变量名。当然，我们也顺便聊了聊有关「中文变量」与「中文编程语言」的区别、在代码中使用中文（非拉丁字符）的一些问题，以及由此引发的许多额外话题。
