Ep 4. 说是要聊「可视化编程」，结果聊了大半天的快捷指令和游戏编程

2021-07-16 podcast

各位 Echo.js 的听众朋友们，我们鸽了一年，终于回来了！ 🙇

本期，我们从越来越火热的可视化编程概念开始，探讨这个只需要拖拖拽拽就能「写代码」的方式到底对我们生活有什么样的影响。省流小助手：对于普通人做做自动化、或是刚刚开始接触编程来说，帮助还是很大的；但它目前对程序员来说，还是不太友好。
